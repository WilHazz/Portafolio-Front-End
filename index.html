APP Del Clima

Esta es una aplicación del clima desarrollada con React, que permite consultar el clima actual de cualquier ciudad mediante el consumo de una API gratuita de clima. Utilizando Hooks de React como `useState` y `useEffect` para el manejo del estado y el ciclo de vida de los datos, junto con CSS puro para el diseño visual de la interfaz.

El propósito principal fue practicar el consumo de APIs en React, el uso de Hooks, y mejorar la experiencia con la creación de interfaces limpias y funcionales.
